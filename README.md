### StarBuzz

- Starbuzz 커피 전문점은 커피와 차를 판매한다. 커피와 차를 준비하는 과정은 각각 다음과 같다.
- 커피와 차를 준비하는 과정을 자바 코드로 구현한다.

- 커피(coffee)
    + 물을 끓인다. 
    + 필터를 활용해 커피를 내린다.
    + 컵에 붓는다.
    + 설탕과 우유를 추가한다.
- 차(tea)
    + 물을 끓인다.
    + 차 티백을 담근다. 
    + 컵에 붓는다.
    + 레몬을 추가한다.
